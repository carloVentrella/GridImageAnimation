Grid of image with animation

This plugin, developed in jQuery, creates the grid fo the image and run the animation chosen by the user.
It is completely configurable, users can choose the size of the cells of the grid, the speed of the animations, the type of animations etc.

You can find the instructions on github : Github - GridImageAnimation

On my personal blog there you can find more about it (italian) : Ketek
